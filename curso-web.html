Diseño de Página Web

Aprende a crear estructuras web y darles estilo profesional usando HTML y CSS para construir sitios funcionales.

¿Qué vas a aprender?

  • • Estructurar el contenido de una página utilizando etiquetas semánticas de HTML5.
  • • Dar estilo, color y diseño a tus páginas mediante hojas de estilo CSS3.
  • • Crear diseños "Responsive" (que se adaptan a celulares y tablets).
  • • Publicar tu primer sitio web en internet de forma gratuita.

Requisitos previos

No se requieren conocimientos previos de programación. Solo necesitas manejo básico de PC (crear carpetas, navegar por internet) y muchas ganas de aprender.
